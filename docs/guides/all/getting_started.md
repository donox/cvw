# Getting Started with the CVW Dashboard

This guide covers the basics of using the CVW membership system.

## Logging In

Go to the site URL and log in with your username and password. If you need to reset your password, contact the administrator.

After logging in you will see the dashboard with links to the areas your role can access.

## Navigation

The top navigation bar shows the sections available to you based on your role:

- **Members** — member records (exec, membership, admin)
- **Programs** — meeting program records (exec, program, admin)
- **Financial** — income and expenses (financial, admin)
- **Exec** — executive console, officers, schedule, action items (exec, admin)
- **Groups / Email** — member groups and email tools (membership, admin)
- **Guides** — this documentation (all users)
- **Website** — opens the public-facing website

## Changing Your Password

Click the 🔑 key icon in the top-right corner of any page to change your password.

## Getting Help

Contact the administrator or refer to the role-specific guides listed on the main Guides page.
